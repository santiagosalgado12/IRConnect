# Documentación del proyecto

Esta carpeta contiene la documentación técnica y académica del proyecto **IRConnect**, desarrollada durante la realización de la tesina final.

# Contenido de la carpeta

Esta carpeta incluye los documentos utilizados para el desarrollo, diseño y presentación del proyecto.

## Diagramas

- **Diagrama de casos de uso**  
  Representa las interacciones entre los distintos tipos de usuarios y las funcionalidades del sistema.

- **Diagrama entidad–relación (BD principal)**  
  Describe la estructura de la base de datos principal del sistema, incluyendo entidades, atributos y relaciones.

- **Diagrama entidad–relación (BD manejador)**  
  Representa la base de datos utilizada para gestionar la comunicación entre el servidor y el dispositivo IoT.

## Informe de tesina

- **Informe tesina final.pdf**  
  Documento completo del proyecto que incluye:

  - Fundamentación del problema
  - Objetivos del sistema
  - Diseño de arquitectura
  - Implementación de hardware y software
  - Diagramas de sistema
  - Pruebas y resultados
  - Apartado de marketing y otras asignaturas
  - Conclusiones del proyecto
