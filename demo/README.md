# Demo del sistema

Esta carpeta contiene material de demostración del proyecto **IRConnect**, incluyendo videos que muestran el funcionamiento del sistema completo.

Las demostraciones incluyen:

- Configuración inicial del dispositivo
- Captura de señales infrarrojas desde un control remoto
- Almacenamiento de señales en el servidor
- Emisión de señales infrarrojas hacia dispositivos electrónicos
- Uso de la interfaz web para controlar dispositivos

# Contenido de la demo

Los videos muestran el flujo completo del sistema:

1. Configuración del dispositivo ESP32
2. Captura de una señal IR desde un control remoto
3. Envío de la señal al servidor
4. Almacenamiento en la base de datos
5. Emisión de la señal hacia un dispositivo

Esto permite verificar el funcionamiento de la arquitectura completa del proyecto, incluyendo hardware, firmware y backend.
